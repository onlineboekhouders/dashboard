Dashboard instellen

Volg deze 3 stappen om het dashboard in te stellen. Dit doe je eenmalig — daarna werkt alles automatisch.

STAP 1
Maak een gratis JSONBin account

JSONBin is een gratis online database. Ga naar jsonbin.io → klik op "Sign Up" → maak een gratis account aan.

STAP 2
Haal je API sleutel op

Na het inloggen op JSONBin: klik rechtsboven op je naam → "API Keys" → kopieer de sleutel die begint met $2b$...

STAP 3
Kies een beheerwachtwoord

Alleen wie dit wachtwoord kent kan het dashboard aanpassen. Het team ziet alles maar kan niets wijzigen.

Wachtwoorden komen niet overeen.
Alles wordt veilig opgeslagen in je eigen JSONBin account.
